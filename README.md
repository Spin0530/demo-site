# TangleID Demo Site

This demo site is designed to illustrate the course site issue an claim to the participants.

## Getting Started

Clone the repository.

```shell
$ git clone git@github.com:TangleID/demo-site.git
```

Install the package dependencies.

```shell
$ cd demo-site
$ yarn
```

Launch the app.

```shell
$ yarn start
```
